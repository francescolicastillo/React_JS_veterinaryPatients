# Patient Manager app with React

This projesct is an App for data management. Add, edit, and remove patients. Track contact details, conditions, and appointments in a clean, responsive UI.

## Demo

https://fc-react-js-veterinary-patients.vercel.app/

## Tech-stack

- React
- JavaScript
- Tailwind CSS
- HTML
- CSS
- Vite
- node
- localStorage

## Project Setup

```sh
npm install
```

### Compile

```sh
npm run dev
```
